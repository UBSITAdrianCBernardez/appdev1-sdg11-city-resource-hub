## Local Setup Instructions
### 1. Open the project folder
```bash
cd appdev1-sdg11-city-resource-hub
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the app
Use either command below:
```bash
npm start
```
or
```bash
npx ng serve -o
```

### 4. Open in browser
If it does not open automatically, go to:
```text
http://localhost:4200
```

## Important Note
If `ng` is not recognized on your computer, that is normal when Angular CLI is not installed globally.
Use:
```bash
npx ng serve -o
```
instead of just:
```bash
ng serve -o
```

## Angular Version Check
After install, verify the version with:
```bash
npx ng version
```
